Более подробные описания и настройки схем предпочтений находятся в файлах AI_<номер схемы>.txt

===================================================================================================

; Схема 1: Брутфорсер.
; Тактика: задавить противника нузкоуровневыми и призывными гильдийскими юнитами, быстро набав критичный перевес.
; Специализация: призыв.
; Приоритеты:
; 1. Призыв в любом виде. Специализация.
; 2. Атака и атакующие втор.навыки героя: Нападение, Стрельба, Вербовка, Удача, Логистика.
; 3. Атака, Скорость, Атакующие навыки, Заклинания урона, доп.существ командира.
; 4. Отстройка + Втор.навык Поместья. Атакующие благословения командира.
; Табу: изучение заклинаний, Сила магии и магические втор. навыки героя.

; Схема 2: Тактик.
; Тактика: получение преимущества во многих направлениях за счет максимально быстрого развития тактической группы навыков.
; Специализация: развитие героя.
; Приоритеты:
; 1. Тактические навыки: Обучение, Наставник, Дипломатия, Некромантия, Глаз орла, Поместья, Мистицизм, Тактика, Интеллект, Вербовка.
; 2. Отстройка, Призыв, Услуги гильдий.
; 3. Защитные параметры, навыки, заклинания урона и призыва командира.
: 4. Знание и Защита героя. Специализация.
; Табу: нет.

; Схема 3: Боевой маг.
; Тактика: победа при помощи заклинаний урона и призыва.
; Специализация: развитие героя.
; Приоритеты:
; 1. Втор. навык Мудрость и заклинания урона.
; 2. Знание и Сила героя, втор. навыки Мистицизм,Волшебство и Школы. Специализация.
; 3. Защита/здоровье, сопротивление и заклинания урона/призыва командира. Заклинания призыва.
; 4. Втор. навыки Сопротивление, Защита, Глаз Орла (баз), Некромантия. Услуги гильдий Стрелков, Головорезов, Чародеев (кроме призыва), Кузнецов.
; Табу: Отстройка и призыв из городов, втор. навыки Поместья и Вербовка.

; Схема 4: Воин.
; Тактика: победа существами, более сильными за счет развития атакующих параметров и втор.навыков героя.
; Специализация: развитие героя.
; Приоритеты:
; 1. Атака и Защита героя, втор. навыки: Обучение, Поместья, Вербовка, Нападение, Защита, Специализация.
; 2. Отстройка и призыв.
; 3. Атака, Защита, Здоровье, Урон, любые способности и любая магия командира.
; 4. Втор.навыки Логистика, Удача, Глаз орла, Стрельба, Боевые машины.
; Табу: изучение заклинаний, Сила магии и магические втор. навыки героя.

; Схема 5: Стрелок.
; Тактика: победа за счет подавления войск противника стрелковыми отрядами, с использованием командира в качестве прикрытия.
; Специализация: призыв.
; Приоритеты:
; 1. Отстройка зданий только до стрелков 4го уровня включительно (для Инферно - гоги), Атака и втор. навык Стрельба героя. Специализация.
; 2. Призыв стреляющих отрядов и услуги гильдий Стрелков и Кузнецов.
; 3. Знание и втор. навыки Дипломатия, Удача, Вербовка.
; 4. Здоровье и защита командира, заклинания командира для стреляющих отрядов.
; Табу: изучение заклинаний, Сила магии и магические втор. навыки героя.

; Схема 6: Созидатель.
; Тактика: максимально быстро отстрить высокоуровневых существ, защищая командира и победить за счет призыва высокоуровневых юнитов.
; Специализация: отстройка.
; Приоритеты:
; 1. Отстройка + Втор.навык Поместья. Специализация.
; 2. Здоровье, защита, сопротивление и защитные способности командира.
; 3. Призыв высокоуровневых существ (умножающий бонус), втор.навыки Вербовка, Нападение, Удача, Тактика, Интеллект.
; 4. Защита и втор.навыки героя (Защита, Сопротивление), защитные заклинания командира.
; Табу: изучение заклинаний, Сила героя, Глаз орла.

; Схема 7: Некромант.
; Тактика: получение преимущества за счет Некромантии и услуг гильдии Некромантов.
; Специализация: призыв.
; Приоритеты:
; 1. Втор навыки Некромантия и Дипломатия, Мудрость базовая. Специализация.
; 2. Услуги гидьдии Некромантов и призыв (не гильдийский). Мистицизм, Для командира-пожирателя отстройка Некрополя и заклинание командира Поднять нежить.
; 3. Втор.навыки Магия Земли, Обучение и Вербовка, Знание и Сила героя, Заклинания Земли 1-3 уровня.
; 4. Втор. навыки Нападение, Защита, Сопротивление, Логистика, Удача. Защита, здоровье, Заклинания урона, проклятий и призыва командира. 
; Табу: магические втор. навыки кроме магии Земли, заклинания кроме указанных.

; Схема 8: Призыватель.
; Тактика: победа за счет призываемых существ (гильдии, заклинания), без затрат на отстройку.
; Специализация: развитие героя.
; Приоритеты:
; 1. Дипломатия, Знание героя, Мистицизм, Мудрость(баз).
; 2. Призывы гильдий. Специализация. Заклинания доп.существ командира.
; 3. Втор. навыки: Магия воздуха, Магия воды. Заклинания: Призыв воздух, Призыв:вода, Молния, Лед. молния, Кольцо льда, Лечение, Сила героя.
; 4. Втор. навыки Обучение и Наставник. Защитные параметры и втор. навыки героя и командира. Благословения командира.
; Табу: Отстройка и призыв из городов, втор. навыки Поместья и Вербовка.

; Схема 9: Солист.
; Тактика: максимальное развитие командира, победа за счет самых сильных заклинаний урона и призыва.
; Специализация: развитие командира.
; Приоритеты:
; 1. Втор. Навыки Магия Земли, Магия Огня, Мудрость, Наставник, Мистицизм, Обучение. Специализация.
; 2. Ударные заклинания Земли и Огня, Антимагия, Призыв:Огонь, Призыв: Земля, Телепорт.
; 3. Втор. навыки Защита, Сопротивление, Волшебство, Логистика. Знание и Сила. Защитные параметры командира. Заклинания урона/призыва/лечения командира.
; 4. Любые навыки, способности и магия командира.
; Табу: Отстройка, призыв, услуги Гильдий, все заклинания и навыки кроме указанных.

; Схема 10: Заклинатель.
; Тактика: победа существами, усиленными магией героя и командира.
; Специализация: Отстройка.
; Приоритеты:
; 1. Призыв, отстройка, втор. навыки: Обучение, Поместья, Вербовка, Нападение, Защита. 
; 2. Сила и Знание героя, втор. навыки Мудрость, Мистицизм. Благословения. Специализация.
; 3. Втор.навыки Школы магии.
; 4. Здоровье, магия доп.существ и благословения командира.
; Табу: магия урона, проклятия, Глаз Орла.

; Схема 11: Маг Воздуха.
; Тактика: специализация на одной школе магии и упор на отстройку одного города.
; Специализация: Отстройка.
; Приоритеты:
; 1. Школа Воздуха, Заклинания воздуха. Специализация.
; 2. Отстройка Башни, Некрополя или Цитадели. Призыв (умножающий бонус). Втор. навыки Поместья, Вербовка, Мудрость, Мистицизм.
; 3. Знание и Сила героя, Отстройка других городов.
; 4. Атакующие параметры и навыки командира, Любая магия командира.
; Табу: Навыки и заклинания непрофильной Школы, Глаз орла, боевые параметры героя.

; Схема 12: Маг Воды.
; Тактика: специализация на одной школе магии и упор на отстройку одного города.
; Специализация: Отстройка.
; Приоритеты:
; 1. Школа Воды, Заклинания воды. Специализация.
; 2. Отстройка Оплота, Подземья или Крепости. Призыв (умножающий бонус). Втор. навыки Поместья и Вербовка, Мудрость, Мистицизм.
; 3. Знание и Сила героя, Отстройка других городов.
; 4. Атакующие параметры и навыки командира, Любая магия командира.
; Табу: Навыки и заклинания непрофильной Школы, Глаз орла, боевые параметры героя.

; Схема 13: Маг Огня.
; Тактика: специализация на одной школе магии и упор на отстройку одного города.
; Специализация: Отстройка. Специализация.
; Приоритеты:
; 1. Школа Огня, Заклинания огня, Втор.навык Сопротивление, сопротивление командира.
; 2. Отстройка Замка, Инферно или Сопряжения. Призыв (умножающий бонус). Втор. навыки Поместья и Вербовка, Мудрость, Мистицизм.
; 3. Знание и Сила героя, Отстройка других городов.
; 4. Атакующие параметры и навыки командира, Любая магия командира.
; Табу: Навыки и заклинания непрофильной Школы, Глаз орла, боевые параметры героя.
